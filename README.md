# LLM 프롬프팅 실습 사이트

Google AI Studio의 Gemini API를 사용한 간단한 LLM 프롬프팅 실습 웹 애플리케이션입니다.

## 기능

- ✅ Google AI Studio API Key 입력 및 저장
- ✅ 다양한 Gemini 모델 선택 (Flash, Flash-8B, Pro 등)
- ✅ 프롬프트 입력 및 실시간 응답 확인
- ✅ 에러 발생시 모달 창으로 알림
- ✅ LocalStorage를 통한 API Key 및 모델 선택 저장

## 사용 방법

1. [Google AI Studio](https://aistudio.google.com/app/apikey)에서 API Key 발급
2. 웹사이트에서 API Key 입력 후 "저장" 버튼 클릭
3. 원하는 모델 선택
4. 프롬프트 입력 후 "전송" 버튼 클릭

## 지원 모델

- Gemini 1.5 Flash
- Gemini 1.5 Flash-8B
- Gemini 1.5 Pro
- Gemini 1.0 Pro

## 기술 스택

- HTML5
- CSS3
- Vanilla JavaScript
- Google Gemini API

## 주의사항

- API Key는 브라우저의 LocalStorage에 저장됩니다.
- API Key는 절대 다른 사람과 공유하지 마세요.
- 공용 컴퓨터에서는 사용 후 브라우저 데이터를 삭제하세요.
